# Remote PC Control

A Python Flask app to remotely shutdown, restart, lock, or sleep your PC with a timer and system tray icon.

## Features

- Shutdown / Restart / Lock / Sleep with timer
- Countdown timer visible on webpage
- Cancel button to stop timer
- Runs in system tray without opening CMD
- Automatically opens web browser

## Requirements

- Python 3.10+
- Flask
- pystray
- pillow

## Installation

```bash
pip install flask pystray pillow
